WEB RENEWAL

Project Date May 10th, 2020
Role Identity, Web ReDesign
Client unslate
Visit https://blog.naver.com/rkdqja79

Problem

클리닝마스터의 기존 사이트는 게시판 위주의 웹사이트 형식에 메인홈 화면이 눈이 찌푸려질정도로 낮은 화질의 사진을 사용하여 불편함을 줍니다.

Solution

1. 디자인은 화면에 가득 찬 느낌으로 큼직큼직하게 배치하고 고급스럽고 깔끔하게 함
2. 메인 화면에 포함해야 할 UI
3. 화면 구성마다 서비스를 바로 갈 수 있는 버튼이나 링크가 존재해야 함 : 자세히보기, 더보기 등 단, 버튼이나 링크의 디자인은 일관성 있게 함
4. 제공하는 서비스를 짐작할 수 있도록 화면의 메인이미지를 디자인해야 함
5. 폰트, 여백, 컬러가 일관성 있고 트렌디해야 함
6. 선택한 이미지의 화질이 높고 배너 광고의 경우 직접 제작해야 함
7. 아이콘 스타일, 선굵기, 반복되는 색상 등은 일관성 있게 해야 함
